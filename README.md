## Project Goal

<p style="text-align: justify">A web solution for Gold Property - AU where users can maintain and track property details efficiently using simple interface. The solution is designed to serve its purpose easily and keeping in mind, a similar app that has been used with excel file given below:</p> 

<p align="center">
    <img src="https://i.ibb.co/VH23f4G/Screenshot-2024-01-22-211629.png" alt="GppApp" />
</p>

#### Core Features: 

* User gets access to the dashboard and see property details at a glance
* The platform enables user to add new property details e.g. property name, pricing, images or any other relevant info

## Getting Started

<p style="text-align: justify">Initially the project will restore all the required nuget packages for the project. If not, the following will help to make it done manually. 

For example, to install entity framework and newtonsoft json from nuget, the following command will download the specific version for the project.<p>

### Prerequisites

* Nuget Package Manager

```sh
PM> Install-Package EntityFramework -Version 6.0.0
```

```sh
PM> Install-Package Newtonsoft.Json -Version 10.0.1
```
